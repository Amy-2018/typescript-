## typescript入门

##### TypeScript 的安装和编译

1.   tsc 编译

     npm i -g typescript 

     tsc -v
   
     tsc index.ts
   


2. nodemon编译 

   npm install -g nodemon  
   
   nodemon index.ts 
   

3. IDE自动编译
